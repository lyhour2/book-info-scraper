# book-info-scraper

- **What It Does**
     - Visits multiple pages on the book site
     - Collects book details like:
          - Title
          - Price
          - Availability
          - Rating
     - Saves the data in a clean csv file (`book.csv`)

- **Tools Used**
     - Python 3
     - `requests`
     - `BeautifulSoup` (from `bs4`)
     - `pandas`
